# homebrew-rekody

Homebrew tap for [rekody](https://github.com/tonykipkemboi/rekody) — open-source, privacy-first voice dictation for macOS.

## Install

```bash
brew tap tonykipkemboi/rekody
brew install rekody
```

## Upgrade

```bash
brew update
brew upgrade rekody
```

## Uninstall

```bash
brew uninstall rekody
brew untap tonykipkemboi/rekody
```
